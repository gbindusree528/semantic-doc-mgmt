# Project Requirements - MVP

- User authentication for 4 departments (HR, Finance, Engineering, Safety)
- RBAC for department-specific document access
- Upload PDF, DOCX, and Image files
- Automatic extraction (OCR for images)
- NLP-based classification of documents by department
- Summarization and translation (English-Malayalam)
- Dashboards per department, showing only relevant documents
- Search and filter functions
- Admin interface for managing roles and permissions
